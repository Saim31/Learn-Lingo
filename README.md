# Learn-Lingo
LearnLingo is a Django-based web application for language learning. It allows users to register, access lessons, take quizzes, and practice exercises while tracking their progress. The frontend is built with HTML, CSS, and JavaScript, and the backend uses Python and Django to manage users and content.
